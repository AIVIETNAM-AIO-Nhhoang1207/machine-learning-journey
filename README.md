# 📖 Giới thiệu

Chào mừng bạn đến với **Machine Learning Journey**!

Đây là repository mình tạo ra để lưu lại toàn bộ quá trình học Machine Learning một cách có hệ thống.

Thay vì chỉ đăng các dự án hoàn chỉnh, mình muốn ghi lại cả hành trình học: từ những kiến thức nền tảng, các bài thực hành, notebook, ghi chú cá nhân cho đến các dự án thực tế.

Mục tiêu của repository này là:

* Ghi lại quá trình học tập.
* Ôn tập kiến thức dễ dàng.
* Theo dõi sự tiến bộ theo thời gian.
* Xây dựng một portfolio thể hiện quá trình phát triển bản thân.
